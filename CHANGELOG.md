# Changelog

## 1.1.0 — 20 July 2026

### Added

- GitHub-style light, dark, and automatic themes.
- Syntax highlighting for markup, PHP, CSS, and JavaScript.
- Accessible copy button, optional language label, line numbers, line wrapping, and maximum-height controls.
- Joomla Web Asset Manager integration.
- English and Turkish language files.
- Joomla update server and administrator changelog metadata.

### Fixed

- Preserved PHP opening tags when normalising an editor-escaped nested `<pre><code>` block.
- Added full day-month-year release date and GPL notice to the extension manifest.
